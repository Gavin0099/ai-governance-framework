# v1.0.0-alpha GitHub Release 草稿

`ai-governance-framework` `v1.0.0-alpha` 是第一個把 AI coding runtime-governance spine、external domain seam、trust-signal publishing 與 release-package publishing 一起收進對外描述的 alpha 版本。

說明：
- 這份文件用作 `v1.0.0-alpha` 的 GitHub Release body draft。
- 它保留為歷史定位文件，不代表 `main` 的最新能力。

## 亮點

- 建立 runtime governance spine：`session_start`、`pre_task_check`、`post_task_check`、`session_end`
- 建立 proposal-time guidance、evidence-aware validation、review-facing change-control artifact 的基本骨架
- 建立 external domain contract seam，示例包含：
  - `USB-Hub-Firmware-Architecture-Contract`
  - `Kernel-Driver-Contract`
  - `IC-Verification-Contract`
- 建立 trust-signal publishing：
  - CI / artifact-style bundle output
  - repo-local generated status path：`docs/status/generated/`
- 建立 release-package publishing：
  - repo-local generated release path：`docs/releases/generated/`

## 建議驗證步驟

1. `pip install -r requirements.txt`
2. `python governance_tools/quickstart_smoke.py --project-root . --plan PLAN.md --contract examples/usb-hub-contract/contract.yaml --format human`
3. `python governance_tools/trust_signal_overview.py --project-root . --plan PLAN.md --release-version v1.0.0-alpha --contract examples/usb-hub-contract/contract.yaml --format human`
4. `python governance_tools/trust_signal_snapshot.py --project-root . --plan PLAN.md --release-version v1.0.0-alpha --contract examples/usb-hub-contract/contract.yaml --publish-docs-status --format human`
5. `python governance_tools/trust_signal_publication_reader.py --project-root . --docs-status --format human`
6. `python governance_tools/release_package_snapshot.py --version v1.0.0-alpha --publish-docs-release --format human`
7. `python governance_tools/release_package_reader.py --version v1.0.0-alpha --project-root . --docs-release --format human`
8. `python governance_tools/release_package_publication_reader.py --project-root . --docs-release-root --format human`
9. `python governance_tools/release_surface_overview.py --version v1.0.0-alpha --format human`
10. `bash scripts/verify_phase_gates.sh`

## 必須同時揭露的限制

這個 alpha 是 bounded alpha，不是 fully closed runtime-governance system。

- workflow interception coverage 仍為 partial
- domain enforcement 仍為 mixed / advisory-first
- semantic verification 仍不是 full policy engine
- external contract seam 仍是 versioned contract / plugin marketplace 的前置形態
