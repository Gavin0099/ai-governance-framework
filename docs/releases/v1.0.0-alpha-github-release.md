# v1.0.0-alpha GitHub Release Draft

`ai-governance-framework` 在 `v1.0.0-alpha` 第一次達到可執行的 AI coding runtime-governance framework prototype 狀態。

注意：

- 這份文件是 `v1.0.0-alpha` 當時的 GitHub release body draft
- 它描述的是 alpha 發版時的公開敘事
- 不等於今天 `main` 分支的全部能力

## Highlights

- runtime governance spine 已可運作，涵蓋 `session_start`、`pre_task_check`、`post_task_check`、`session_end`
- proposal-time guidance、evidence-aware validation、review-facing change-control artifact 已進入 framework 主線
- external domain contract seam 已在三個低階領域被實際演練：
  - `USB-Hub-Firmware-Architecture-Contract`
  - `Kernel-Driver-Contract`
  - `IC-Verification-Contract`
- trust-signal publishing 已同時支援：
  - CI / artifact-style bundle output
  - repo-local generated status path：`docs/status/generated/`
- release-package publishing 也已有穩定的 repo-local generated path：`docs/releases/generated/`

## 當時已知邊界

這版 alpha 仍刻意保持 bounded：

- workflow interception coverage 仍然 partial，尚未 fully closed
- 多數 domain enforcement 仍是 mixed / advisory-first，不是 universally hard-stop
- semantic verification 雖已前進，但仍淺於 full policy engine
- external contract seam 雖然真實存在，但還不是 versioned plugin marketplace
