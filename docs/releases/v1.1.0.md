# v1.1.0

發版日期：2026-03-22

`v1.1.0` 是目前最後一個正式對外 release。它的歷史位置是：把 framework 從早期 alpha prototype 往前推到可驗證的 external adoption path，並完成當時的 token-optimization 主線。

注意：

- 這份文件描述的是 `v1.1.0` 發版當下的正式 release-facing 狀態
- `main` 分支後來新增的 runtime / closeout / advisory / status hardening，不應回寫成這版 release 已包含的能力

## v1.1.0 當時完成了什麼

### 1. Cross-platform adoption toolchain

- `adopt_governance.py` 取代 `bash scripts/init-governance.sh`，成為 canonical adopt entrypoint
- Windows adoption path 正式成立
- `--refresh` 可重算 baseline hash 與 section inventory
- `--dry-run` 可預覽寫入
- framework root 可透過 env 或 upward scan 自動發現

### 2. Governance drift 與 schema visibility

- drift checker 當時已達 16-check 規模
- fresh-adopt repo 在 baseline、placeholder、template-copy guard、inventory freshness 上更可見
- `minimum-legal-schema.md` 已被拉到 onboarding touch point
- `AGENTS.md` baseline 改成 plain `N/A`，減少新 adopt repo 因模板字樣而誤報 drift

### 3. Token optimization roadmap

當時完成了完整七步 token payload optimization：

- Step 1 baseline：約 `104,696` tokens
- Step 7 result：約 `49,202` tokens
- output tier 分離：Tier 1 / Tier 2 / Tier 3
- `L0` / `L1` / `L2` task-level detection
- summary-first contract loading
- payload audit logger 與 onboarding audit lane

### 4. Runtime hardening

- onboarding 有自己獨立的 audit lane
- Windows terminal 對不可編碼 Unicode 採安全 fallback
- `session_start.py` 與 `pre_task_check.py` 開始帶 task-level-aware tier output

### 5. Authority 與 rule classification

- `governance/AUTHORITY.md`
- `governance/RULE_REGISTRY.md`
- `governance_tools/authority_loader.py`
- `governance_tools/rule_classifier.py`

這一版讓 authority / rule pack classification 變得比較可機器消費。

### 6. Test coverage

當時 release note 記錄的是：

- `1,386 tests` at `v1.1.0`
- 相較 `v1.0.0-alpha` 的 `835` tests 明顯成長

## v1.1.0 當時刻意保留的邊界

這一版 release 當時仍刻意保留：

- workflow interception coverage 仍不完整
- domain enforcement 多數仍是 mixed / advisory-first，不是 universally hard-stop
- semantic verification 仍淺於 full policy engine
- external reviewer onboarding 仍是下一個重要 Beta gate

換句話說，`v1.1.0` 的價值不是宣稱 everything solved，而是把當時最重要的 adopt / runtime / release-facing 基礎盤起來。
