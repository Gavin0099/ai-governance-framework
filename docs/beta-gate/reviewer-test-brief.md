# Reviewer Test Brief — Beta Gate Condition 2

> 狀態：ready for use
> 建立：2026-03-30
> Gate condition：human self-serve onboarding without author guidance

---

## 目的

這份 brief 定義的是測試條件，不是 framework 教學文件。

不要在 reviewer 開始前把這份文件給 reviewer。  
它主要用來：

- 設定測試條件
- 在跑完後做 debrief

---

## Reviewer 起始條件

reviewer 只會拿到：

- repo URL
- 一句簡短 framing
- 時間預算
- 不向作者提問的前提

reviewer 不會先拿到：

- file pointer
- hidden checklist
- 作者導讀

---

## 一句話結論

這份 brief 的目的，是保證 Beta Gate condition 2 真的是 human self-serve onboarding 測試，而不是作者先行導覽後的半引導 run。
