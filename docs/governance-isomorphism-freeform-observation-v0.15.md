- as_of: 2026-05-13
- phase: discovery

- scope: adversarial boundary probe for governance abstraction

## Observation Discipline

This document is intentionally pre-taxonomy.
Observed degradation patterns are recorded before invariant classification stabilizes.

The purpose is boundary discovery,
not capability expansion,
not governance productionization,
and not ontology locking.

Taxonomy is treated as hypothesis, not truth structure.

## Method (Freeform)

Each case records only:
- observed break
- why current invariant set is insufficient
- downgraded claim
- residual uncertainty

No canonical class is assigned at this stage.

## Freeform Degradation Observations

### Case 01: Manager override under nominal authority chain

- observed break:
  Formal authority graph exists, but a manager-side informal override changes outcome without policy-state transition.
- why current invariant set is insufficient:
  `authority explicitness` captures graph visibility, but not social legitimacy transfer that bypasses formal edges.
- downgraded claim:
  From "authority trace is complete" to "formal authority trace is partial; informal legitimacy path not captured."
- residual uncertainty:
  Unclear whether this is persistent organizational property or local team behavior artifact.

### Case 02: Cross-team escalation with discretionary reroute

- observed break:
  Escalation path is rerouted via social channel instead of deterministic `if X -> Y` path.
- why current invariant set is insufficient:
  `escalation determinism` identifies mismatch but does not represent discretionary reroute as first-class governance state.
- downgraded claim:
  From "escalation path is deterministic" to "escalation intent is observable; path execution is non-deterministic."
- residual uncertainty:
  Unknown if reroute was efficiency optimization or governance bypass.

### Case 03: Sales exception handling under contested evidence

- observed break:
  Evidence hierarchy is overridden by commercial priority in final decision context.
- why current invariant set is insufficient:
  `evidence legitimacy` assumes source-bounded acceptance, but cannot encode negotiated weighting by organizational incentives.
- downgraded claim:
  From "evidence-bound decision" to "evidence-informed decision with socially mutable weighting."
- residual uncertainty:
  Unknown whether this behavior is bounded exception policy or latent default in high-pressure windows.

## Interim Reading

- Partial structural isomorphism remains plausible at concept surface.
- Degradation appears where social dynamics become execution-relevant.
- Deterministic governance invariants may be substrate-bound rather than organization-universal.
